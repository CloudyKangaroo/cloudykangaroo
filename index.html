Cloudy Kangaroo

The Cloudy Kangaroo project is an interface to allow you to interact with your management systems. Currently Puppet, Sensu and Ubersmith are supported data providers/integration points.

Integrations and data providers like Graphite, Libvirt, Salesforce, osTicket, ZenDesk are also in the works.

Community

Contegix is a Managed Services Provider in St. Louis, MO. The project originated within their walls as a way to tie Puppet and Sensu into their Helpdesk, Support and CRM systems. Currently Logstash, Elasticsearch, Kibana and Graphite are being deployed and integrated.

Contegix is committed to supporting and maintaining this product as Open Source Software and it's use by others is encouraged.

Current Feature Set

Tickets

  • Create New Ticket
  • View Tickets by Customer, Device, or Queue

Monitoring

  • View Events
  • Silence Events, temporarily
  • Assign Events to Tickets (new or existing), permanently silencing them
  • Resolve Events

Puppet

  • View Facts
  • Assign Hiera Configuration
  • Purge Hosts
  • See Puppet/PuppetDB Health
  • View Report status, last report
  • View recent failed resources

Authors and Contributors

In September 2013, Jonathan Creasy (@johann8384) set out to build an interface for our support and engineering teams that would bring a modern and functional interface to our new ticketing system, as well as bring integration points for our monitoring and configuration management systems.

In December, the initial release was deployed and Rich Chatterton (@rmc3) began his involvement in the project to support improved interface design and added some back-end integration features.

In the coming months additional contributors are expected both from within Contegix and from within two other organizations that are currently working with this system.

Roadmap
